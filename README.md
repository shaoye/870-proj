# 870-proj

## Environment

- python 3.7+


## Install

> On macOS and Linux:

```
python3 -m venv env

source env/bin/activate

python3 -m pip install -r requirements.txt
```

> On Windows:

```
python -m venv env

.\env\Scripts\activate

python -m pip install -r requirements.txt
```

## Run
```
python3 ./model1.py
```
